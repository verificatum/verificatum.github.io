verificatum-vtsc

Verificatum TypeScript Cryptography library

Self-contained cryptographic library for use in electronic voting clients. Complete documentation with references to the literature on basic arithmetic algorithms makes it good teaching material.

Installation

Dependencies

 
# Install building tools.
sudo apt install make m4

# Most recent version of node.
curl -fsSL https://deb.nodesource.com/setup_current.x | sudo -E bash -
sudo apt install -y nodejs

# TypeScript globally.
sudo npm install -g typescript

# TypeScript definitions for NodeJS locally.
npm install -D @types/node

# webpack if you intend to generate a JS-bundle for browser.
npm install -D webpack webpack-cli webpack-node-externals

# Install ESLint for use with TypeScript.
npm install -D eslint @typescript-eslint/parser @typescript-eslint/eslint-plugin

# Install TypeDoc to update the generated documentation.
npm install -D eslint-plugin-tsdoc typedoc

Targets

To make it easy to understand the project, the repository contains the generated TypeScript code and the generated documentation. We refer the user to the makefile targets to generate JavaScript code for Node, WebAPI, or bundled using Webpack.

Contributing

The source code resides in the src/ts directory. The TypeScript code is itself generated using M4. If you open an issue, then please make sure that you open it for the source and not the generated TypeScript code.
