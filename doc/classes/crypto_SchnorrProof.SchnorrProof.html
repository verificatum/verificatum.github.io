Class SchnorrProof

Sigma proof of a pre-image of a homomorphism from a ring to a group using a generalized Schnorr proof. More precisely, if Hom : R -> G is a homomorphism, where R is a product ring of a finite field Z/qZ of order q, and every non-trivial element in G has order q, then the protocol is defined as follows on common input x and private input w such that (x, w) is in the NP relation.

  1. Prover chooses a in R randomly and computes A = Hom(a).
  2. Verifier chooses a random challenge v in Z/qZ.
  3. Prover computes a reply k = w * v + a in R.
  4. Verifier accepts if and only if x^v * A = Hom(k), where the product is taken in G.
param homomorphism

Underlying homomorphism.
