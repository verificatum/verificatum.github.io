Class EC

Raw container class for elliptic curves.

ASSUMES: 0 <= a, b, gx, gy < modulus, n > 0, and x^3 + b * x + a (mod modulus) is a non-singular curve of order n.

param modulus

Modulus for underlying field.

param a

First coefficient for curve of Weierstrass normal form.

param b

Second coefficientfor curve of Weierstrass normal form.

param n

Order of elliptic curve.
