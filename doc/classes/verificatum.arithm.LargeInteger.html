Class LargeInteger

Class for large immutable integers that handles memory allocation and provided utility functions.

param first

Can be: (1) sign of explicit integer, (2) bit length of random integer, (3) byte array containing the bits of an integer, (4) hexadecimal representation of integer, (5) byte tree representation of integer, or (6) Javascript "number" representation of integer.

param second

Can be: (1) value of explicit integer, (2) or source of randomness, or in cases (3)-(6) it must be empty.
