Class ModPowProd

Pre-computes values to be used for simultaneous exponentiation for a given list b of k bases and a modulus m. The method ModPowProd.modPowProd then takes a list of exponents e and outputs the modular power product

g[0] ^ e[0] * ... * g[k - 1] ^ e[k - 1] mod m.

The number of exponents must match the number of bases for which pre-computation is performed.

param bases

List of bases.

param modulus

Modulus.
