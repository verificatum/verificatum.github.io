Test Suite

Tests should be executed on the target platform in addition to the development platform, due to the heterogeneity of platforms, engines, and versions. Thus, the tests corresponding to running make check_files can be run directly in the browser.

  • Test Suite

Benchmark Suite

Similarly, running times should be estimated on the target platform for clients to be realistic. Thus, the benchmark suite corresponding to running make bench_files can be run directly in the browser.
